GTA VI Title ID Found In Xbox API — The Pre-Order Button Is Almost Ready

Source: Reddit user u/BlackAnt02 via r/GTA6. All ID data reproduced from the original thread. 56ViceLane independently verified the Package Name against publicly accessible Microsoft Store API endpoints. This is not official Rockstar or Microsoft communication.

Somewhere inside Microsoft's backend, a string of characters has been sitting quietly, waiting for the right person to find it. On May 28, 2026, that person was a Reddit user named BlackAnt02. What they found isn't a rumor, isn't a leak from an inside source, and doesn't require a single anonymous tip to verify. It's a Title ID — and it means Rockstar Games has already registered GTA VI on Xbox as a purchasable product.

The pre-order button isn't live yet. But the infrastructure to flip it on? That's done.

Who Is BlackAnt02

BlackAnt02 is a member of the r/GTA6 community with a history of digging into storefront APIs and backend product databases — the kind of methodical work that doesn't get the headlines that trailer breakdowns do, but frequently delivers harder evidence. This is not a screenshot of a DM from someone's "friend at Rockstar." This is public-facing API data pulled from Microsoft's own product catalog.

The original Reddit post was straightforward: the IDs, the Package Name, and the observation that this represented active product registration. The GTA6 community picked it up fast. The thread gained traction because the data was verifiable — you can look up Xbox Title IDs in Microsoft's public endpoints yourself. BlackAnt02 didn't need to be believed on faith. The numbers speak for themselves.

What A Title ID Actually Means

Title IDs are assigned by Microsoft when a developer formally registers their title in the Xbox developer program. It is not a placeholder. It is not reserved speculatively. A Title ID exists because a publisher — in this case, Rockstar Games under the Package Name RockstarGamesInc.28872443D20A7_k2ss7vbb13xte — submitted their product for registration and it was processed.

The decimal ID 2035456122 and its hex equivalent 7952987A are two representations of the same registered product identity. Think of it like a UPC code for the Xbox ecosystem. Before a game can appear in the Microsoft Store — before it can be wishlisted, before it can accept a pre-order, before it can show a price — it needs this ID. That work is already done.

"Before a game can appear in the Microsoft Store — before it can be wishlisted, before it can accept a pre-order — it needs this ID. That work is already done."

The Xbox Product ID 9NL3WWNZLZZN is the storefront-facing identifier. If Microsoft flips the listing live tonight, this is the string that would populate the URL for GTA VI's Xbox store page. Everything is staged. The only thing missing is a date when Rockstar tells Microsoft to make it public.

The PlayStation Side

Two PlayStation Store IDs surfaced in the same thread: PPSA01547_00 and PPSA29660_00. On PlayStation, a single game title can carry multiple IDs — this typically indicates different editions, regional variants, or digital versus physical distinctions. The fact that two IDs exist is worth paying attention to.

Earlier leaks suggested GTA VI will launch with at least six distinct editions — the DetectiveSeeds leak from May included three standard tiers, a collector's edition, and two PS5-specific bundles. Two PlayStation product IDs aligns with a tiered release structure. This is not two separate games. This is how modern AAA titles with multiple purchase options get registered in Sony's backend.

PlayStation's PPSA product format is used for PS5 native titles. Both IDs carrying the PPSA prefix confirms both registrations are for current-generation hardware — consistent with everything Rockstar has communicated about the game's platform targets.
